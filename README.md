# FILM-REVIEW-AND-TICKET-BOOKING-SYSTEM
The Grand Imperial: A Full-Scale Interactive Cinema Portal  A sophisticated, vintage-themed web application built with Vanilla JavaScript, HTML5, and CSS3. This project replicates a premium 1920s cinema experience, featuring a dynamic movie database, a functional seat booking system, a film trivia engine, and a loyalty rewards program.

A sophisticated, interactive movie review and ticketing portal designed with a **1920s Art Deco vintage aesthetic**. This project blends the "Golden Age" of cinema with modern web functionality.

---

## ✨ Key Features

### 🎞️ Interactive Critique Gallery
Browse a curated collection of classic and modern films. Includes a **real-time search bar** and **genre-based filtering** (Drama, Sci-Fi, Noir, Action).

### 🎟️ Virtual Box Office & Seat Selector
Experience a functional booking simulation. Users can:
* View current showtimes and upcoming premieres.
* Select seats from a dynamic visual grid.
* Book **VIP seating** with automatic price calculations.

### 🧠 Cinema Trivia Engine
Test your knowledge with an integrated film quiz. Features a **progress tracker**, instant feedback, and a score management system.

### ⭐ Imperial Membership (Loyalty System)
An automated rewards program where users earn points for booking tickets, writing reviews, and passing quizzes. Watch your status grow from **Silver** to **Platinum**!

### 🌓 Dual-Theme Experience
Switch between the "Vintage Dark" mode and "Classic Light" mode with a single click to suit your viewing preference.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Advanced Grid & Flexbox)
* **Logic:** Vanilla JavaScript (ES6+)
* **Typography:** Google Fonts (Playfair Display, Rye, Special Elite)
* **Animations:** Custom CSS Transitions and Keyframes
